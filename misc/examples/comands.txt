Upload script:

you need:
	login:password
	file name
	url
	path to file

curl -i -u [login:password] -H "file=[file name]" -X POST [url] -F file=@"[path]"

for example:
curl -i -u admin:admin -H "file=plik.apm" -X POST http://localhost:5502/bin/apm/cqsmFileUpload -F file=@"C:/Users/kamiljul/Desktop/plik.apm"
